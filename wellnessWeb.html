NBCC Wellness Web Application

Login Page

Index Page - a home page featuring latest content and upcoming events.

Articles Page - articles posted by admins appear here for users to read.

Resolutions Page - users can check off resolutions added by admins.

User Dashboard Page - users can track their progress as they make their way through all the content.

Support Page - users can access support services

User Settings page - where users can modify their profile settings. Note all forms are collapsible. Here I have opened two of them for demonstration purposes.

Admin pannel - available only to admins. Here admins can create new content, as well as edit or delete existing content. A notificaiton is sent only when new content is created.

New Content page - when an admin creates new content they do so on this page.

User Progress Page - admins can monitor the progress of each user.
